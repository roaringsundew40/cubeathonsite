How to play Cube-A-Thon

November 21, 2020

To play Cube-A-Thon you will download the game through itch.io. itch.io is a site for game hosting. You can go there and get a secure download. Continue Reading to get directions based on your Operating System.

Before continuing open a browser of your choice and go to this link

For Windows: Find the Windows Download. It should be called Cubeathon Installer. Click Download. It will download a .msi file. If it gives you a warning saying this file may be dangerous just ignore it. Run the .msi file after download. Follow the steps through the installer and then once you have finished installing search for Cube-A-Thon.exe. Once you find it open it. Then click play. Use the "A" and "D" keys to move around.

This game is currently not avalible for any other Operating Systems.
